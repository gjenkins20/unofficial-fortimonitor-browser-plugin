Required for the Add Fabric Connection and Server Lookup tools. Get a Read/Write key from your FortiMonitor account settings. Stored locally in your browser via chrome.storage.local; never transmitted except to api2.panopta.com.

Test verifies the key has at least read access. RW capability is verified only on actual writes.

Required for the Ask Claude tool (Beta). Create a key at console.anthropic.com. Stored locally in your browser via chrome.storage.local; sent only to api.anthropic.com. Cost warning: calls go directly to Anthropic with your key - no rate limit or budget cap is enforced by this extension.

Ask Claude is an in-plugin Claude chat that queries your FortiMonitor environment via a curated set of v2 API tools. Shown by default; toggle off to hide the tile and the Anthropic API key settings section. Requires an Anthropic API key in addition to your FortiMonitor RW key.

Search Servers pages the full /server list and filters by a specific attribute (e.g., Model = FGT60F). Pick the attribute from your tenant's catalog, enter the value, exports matches as CSV. Read-only. Hidden by default.

Off by default. When on, the toolkit adds an FM Toolkit entry to FortiMonitor's left sidebar as an alternative way to open this popup. The Chrome toolbar icon (above) is the primary entry and is always available regardless of this setting.

On by default. When on, each visible UI feature this extension adds to FortiMonitor pages carries a small diagonal FM Toolkit ribbon in its top-right corner, so you can tell at a glance which controls come from this extension vs. native FortiMonitor. The ribbons are click-through and persist for the page lifetime. Toggle off to hide them; the change takes effect on the next page load.

Reorder or hide the columns this plugin adds to FortiMonitor pages. Each augmentation has its own list. Changes apply immediately and persist across sessions. You can also drag the sub-headers directly on the FortiMonitor page itself.

Off by default. When on, scan-failure and API-error screens include diagnostic fields - HTTP status, final URL after redirects, content-type, and a short redacted response-body preview - plus a Check session button that probes FortiMonitor and reports XSRF-cookie presence. Intended for troubleshooting installs where your browser session is not being recognized.
