Required for the Add Fabric Connection and Server Name Lookup tools. Get a Read/Write key from your FortiMonitor account settings. Stored locally in your browser via chrome.storage.local; never transmitted except to api2.panopta.com.

Test verifies the key has at least read access. RW capability is verified only on actual writes.

Off by default. When on, scan-failure and API-error screens include diagnostic fields — HTTP status, final URL after redirects, content-type, and a short redacted response-body preview — plus a Check session button that probes FortiCloud and reports XSRF-cookie presence. Intended for troubleshooting installs where your browser session is not being recognized.
