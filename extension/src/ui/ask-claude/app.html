Heads up: usage notes + related projects

Cost warning. When the provider is Anthropic, this tool calls the Anthropic API directly with your key. There is no rate limit or budget cap on our side - cost exposure is on your account. Keep conversations short and close this tab when you're done.

Local provider. Chat turns route to the local OpenAI-compatible server you configured in Settings. No data leaves your network and there is no per-turn cost; throughput is bounded by your local model.

Tool-capable models only. Each chat turn sends function definitions and expects the model to emit tool_calls. Models without tool training (Gemma 1/2/3, Llama 2, base Mistral) ignore the tools and reply with text. Pick from qwen2.5, llama3.1, mistral-nemo, command-r-plus, qwen3, or any other model that lists tools under ollama show <model>'s Capabilities line. Stick to the read-only tier (~10 tools) for 7B/8B models; bigger catalogs reliably need 14B+.

Like this? This is a narrow slice (11 tools). For full 241-tool coverage plus a knowledge layer over the FortiMonitor docs, check out the Unofficial FortiMonitor MCP Server. Runs alongside Claude Desktop or any MCP client.
