FMN-154 / FMN-155 / FMN-156 - Reports-page cards + extension UI preview

Three new toolkit features rendered (a) as cards on the FortiMonitor Reports page, sharing the FMN-86 "FM Toolkit" attribution ribbon, and (b) as their in-extension tool UI. The native FortiMonitor tile style is reconstructed from the existing mockups in this folder; final pixel-exact match will land when augment.js gates on the live Reports path.

What you're evaluating: tile copy, tile primary/secondary actions, the relative weight of the FM Toolkit ribbon, and the diff / composer surface area. Pixel-level chrome (fonts, borders, exact icon shape) will be cross-checked against the live tenant before implementation begins.
