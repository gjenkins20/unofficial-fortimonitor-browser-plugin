Scenarios:

Reorder or hide the columns this plugin adds to FortiMonitor pages. Each augmentation has its own list. Changes apply immediately and persist across sessions.

Instances list /report/ListServers
How it works:
• Drag (⋮⋮ handle or anywhere on the row) reorder by dropping above or below another row. Blue line shows the drop target.
• ↑ / ↓ buttons same reorder action via keyboard / accessibility.
• Eye toggles visibility. The Instance label is locked visible (it's the FortiMonitor-native column - hiding it would orphan the row's link).
• Reset restores the default order with all columns visible.
• On the page itself: the sub-headers (Instance / IP Address / DNS Name) are also draggable. Try dragging "DNS Name" to the left of "IP Address" in the table on the right. Blue side-line shows the drop target.
• Storage: chrome.storage.local["webguiColumns"]["instances-ip-dns-columns"].
